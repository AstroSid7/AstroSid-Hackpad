\# AstroSid's Macropad ! - 6 Keys + 1 Knob



This is a custom mechanical macropad designed for the Hackpad project. It features 6 switches and 1 rotary encoder for media control, powered by a \*\*Seeed XIAO RP2040\*\*.



\## 🚀 Features

\* \*\*6 Mechanical Switches:\*\* Mapped to Play/Pause, Skip, and Mute.

\* \*\*Rotary Encoder:\*\* High-precision volume control with a click-to-mute function.

\* \*\*KMK Firmware:\*\* Easy-to-edit Python-based firmware.

\* \*\*Compact Case:\*\* 3D-printable 2-part design.

\## 🛠️ Bill of Materials (BOM)

| Quantity | Component | Description |

| :--- | :--- | :--- |

| 1 | Seeed XIAO RP2040 | Main Microcontroller |

| 6 | Cherry MX Style Switches | Mechanical Keys |

| 1 | EC11 Rotary Encoder | For scrolling/volume |

| 6 | 1N4148 Diodes | For the key matrix |

| 1 | 3D Printed Case | Top Plate \& Bottom Shell |



\## 📂 Folder Structure

\* `/CAD`: Contains the `.STL` files for the case.

\* `/PCB`: Contains the KiCad project and PCB design.

\* `/Firmware`: Contains the `main.py` KMK script.

\* `/production`: Contains the `gerbers.zip` and final 3D files.
